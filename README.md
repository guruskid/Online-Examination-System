Installation Steps
1. Download zip file and Unzip file on your local server.
2. Put this file inside "c:/wamp/www/" .
3. Database Configuration
Open phpmyadmin
Create Database named quiz_new.
Import database quiz_new.sql from downloaded folder(inside database)
4. Open Your browser put inside "http://localhost/Online_exam_New/"
5. To Login as admin put "http://localhost/Online_exam_New/admin"
